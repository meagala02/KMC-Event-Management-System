# KMC Event Management System (SOC Module Assignment)

A Full-stack Event Management System built using **.NET Framework**.

## Project Structure
- **KMC_API:** RESTful Web API for backend data handling.
- **KMC_ClientApp:** Frontend application for user interaction.
- **KMC_System:** Core business logic and database management.

## Tech Stack
- **Language:** C#
- **Framework:** ASP.NET Web API
- **Database:** SQL Server
- **IDE:** Visual Studio
